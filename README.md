# 2047 Technologies Website

A static, GitHub Pages-ready company website for **2047 Technologies**, a Zimbabwe-based IT consultancy focused on practical websites, custom web systems, data tools, IT support, security awareness, and workflow automation.

## Files

- `index.html` - Main website page and contact form
- `style.css` - Responsive styling and modern tech visual design
- `script.js` - Mobile menu, active navigation, reveal animations, and form submit state
- `assets/logo.svg` - Main 2047 Technologies logo
- `assets/logo-horizontal.svg` - Horizontal logo for banners and documents
- `assets/favicon.svg` - Browser favicon

## Deploy on GitHub Pages

1. Create or open a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save and wait for GitHub Pages to publish the site.

## Customisation notes

- Replace social links in `index.html` when the company gets dedicated social accounts.
- Replace the FormSubmit email when a company email is available, for example `info@2047technologies.co.zw`.
- Update service descriptions as the company adds new proven offerings.
